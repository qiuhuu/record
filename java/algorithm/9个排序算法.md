# 时间复杂度

- O(n2)【平方阶】 排序
	- 直接插入、直接选择、冒泡排序
- O(nlog2n)【线性对数阶】 排序
	- 快速排序、堆排序、归并排序
- O(n1+§)) 排序，§ 是介于 0 和 1 之间的常数
	- 希尔排序
- O(n)【线性阶】 排序
	- 基数排序、桶排序、箱排序
